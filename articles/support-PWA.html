小岚的小窝已支持渐进式 Web 应用（PWA）

PWA 是什么?

PWA:渐进式 Web 应用

PWA 介绍:MDN

为什么要让博客支持 PWA 技术?

优势

PWA 是可被发现、易安装、可链接、独立于网络、渐进式、可重用、响应性和安全的。
如果你忘记了我的网站域名，你可以直接通过已安装的 PWA 应用来打开我的博客。
最重要的是能让我的网站进行离线访问！

如何安装 PWA 应用

支持的浏览器

市场主流浏览器已经能够较好支持这一应用标准
PWA 所需的关键要素是 service worker 支持
所以只要是支持 service worker 的浏览器
理论上均支持 PWA 应用：

  • Chrome（v70 稳定版及以上版本）
  • Firefox（仅支持 Android v58+）
  • Edge
  • Safari（iOS 11.3+）
    等等。

安装应用

图片中使用的均为 Google Chrome 浏览器，不同的浏览器位置可能不同。

电脑：
可在浏览器右上角安装应用

安装好后将会在桌面生成图标

手机：
在浏览器最下面安装应用

或者点击右上角三个点安装

安装好后将会在桌面生成图标
